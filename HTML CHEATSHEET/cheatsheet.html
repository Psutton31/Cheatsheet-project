Pats HTML/CSS Cheatsheet


HTML BASIC ELEMENTS

Tag Name Description
<head> head Contains metadata/information for the document
<body> body Defines the document's body
<header> header Defines a header for a document or section
<section> section Defines a section in a document
<article> article Defines a section in a document
<figure> figure Specifies self-contained content
<form> form Defines an HTML form for user input
<input> input Defines an input control

Media Tags and Attributes

Tag Name Description Attributes
<img> Image display an image: require "src" attribute "src" for directory, "alt" for alt text, height and width(optional)
<audio> audio Play Audio "controls" in opening tag to display controls, <source> must be inside audio element using "src"
<video> video Play Video "controls" to display controls, <source> must be inside Video element using "src", "height" and "width"
